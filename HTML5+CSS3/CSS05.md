# CSS05
## 구조선택자
- 부모의 요소 안에서 특정한 위치의 요소를 선택할 때 사용
- 후손+자손선택자와 함께 많이 사용!
- :first-child 같은 부모아래에서 형제관계에서 첫번째에 해당하는 요소를 선택
- :last-child 같은 부모아래에서 형제관계에서 마지막번쨰에 해당하는 요소를 선택
- :nth-child(n) 형제관계의 순번째에 위치하는 요소를 선택 (1,2,3...), 2n+1(홀수), 2n(짝수)
- :nth-last-child(n) 부모박스 안에서 형제관계 중에 뒤에서 수열번째에 위치하는 태그를 선택
        
## 형태구조선택자
- 일반구조선택자와 비슷하지만 태그 형태를 구분
- :first-of-type 부모박스 안에서 형제관계 중에 첫번째에 위치하는 태그를 선택
- :last-of-type 부모박스 안에서 형제관계 중에 마지막번째에 위치하는 태그를 선택
- :nth-of-type 부모박스 안에서 형제관졔 중에 수열번째에 위치하는 태그를 선택
- :nth-last-of-type 부모박스 안에서 형제관계 중에 뒤에서 수열번쨰에 위치하는 태그를 선택
