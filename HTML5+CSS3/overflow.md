# Overflow
> 부모박스에서 내용물이 넘치는 경우 어떻게 보여질지 결정하는 속성

## overflow: hidden;
> 영역을 벗어나는 부분을 보이지 않게 처리
> height가 존재하지 않을 때는 높이를 대신
> width가 존재할 때 내용물을 안보이게 처리

## overflow: scroll;
> 안에 콘텐츠가 넘치는 경우 스크롤, 내용물이 넘치지 않아도 자동 스크롤 생성

## overflow-x: hidden;
> x축(가로)의 넘치는 내용물을 어떻게 보여질지 결정하는 속성

## overflow-y: hidden;
> y축(세로)의 넘치는 내용물을 어떻게 보여질지 결정하는 속성
