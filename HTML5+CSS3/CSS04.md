# CSS04
## 후손선택자
- 해당요소의 하위요소에 있는 특정한 요소를 모두 선택
- 선택자A 선택자B
## 자손선택자
- 해당요소의 바로 밑에 위치하는 특정한 요소를 선택
- 선택자A>선택자B
## 동위선택자
- 동위 관계에 있는 요소중에 해당요소보다 뒤에 존재하는 특정한 요소를 선택할 때 사용
- 선택자A+선택자B: 선택자 바로 뒤에 위치하는 선택자B를 선택
- 선택자A~선택자B: 선택자A 뒤에 위치하는 선택자B를 모두 선택
## 반응선택자
- 사용자가 마우스를 이용해서 혹은 특정한 행동(반응)을 취했을 때 css속성을 지정할 수 있는 선택자
- hover: 특정한 요소에 마우스를 올렸을 때!
- active: 특정한 요소에 마우스로 클릭을 한 순간!
